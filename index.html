Microsoft Computer Vision API

How-To Guide by Sana Tauqir

Introduction

Microsoft's Computer Vision API is part of their emphasis on Cognitive Services to use gathered data to enable detection of objects (landmarks), celebrities, faces, and even text in images. After analysis this API provides some data as to their features. This enables you to include images in your search results and when implementing tagging techniques. While the Computer Vision API by itself is great there are further Cognitive Services API's worth noting such as Face and Emotion API's which provide in-depth information.
